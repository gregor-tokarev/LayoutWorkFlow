> This is my template for creating static sites. Powered by esbuild(js) and gulp(all other). 
> Features included:
> 1. Images optimization
> 2. Style compression
> 3. Scss compilation
> 4. javascript bundling & optimization
> 5. Webp generation
> 6. Templating
> 7. Favicon generation

## Install

```sh
npm install
```

## Usage

```sh
npm run dev
```
